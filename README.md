## 📊 Descripción

**GastoApp Pro** es una aplicación web progresiva (PWA) para gestionar ingresos y gastos de forma simple, visual y eficiente.

Permite registrar movimientos, visualizar métricas en tiempo real y exportar reportes en PDF, todo sin necesidad de backend.

---

## ✨ Features

- 📈 Dashboard financiero en tiempo real
- 💵 Registro de ingresos y gastos
- 🧠 Cálculo automático de balance y porcentaje utilizado
- 📊 Gráfico dinámico por categorías (Chart.js)
- 📅 Filtro por mes y año
- 📄 Exportación a PDF profesional
- 💾 Persistencia con LocalStorage
- 📱 Progressive Web App (instalable)
- 🎨 UI moderna con TailwindCSS + animaciones

---

## 🧱 Tecnologías utilizadas

- ⚡ HTML5
- 🎨 TailwindCSS
- 📊 Chart.js
- 📄 jsPDF + AutoTable
- 🧠 JavaScript Vanilla
- 📦 Service Worker (PWA)
